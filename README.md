# dend-mod-gui-pyside6
PySide6バージョンの、電車でDのプログラム統合版

## 概要

dend-mod-guiは、電車でDのデータ改造のため、GUI画面上で編集するソフトウェアである。

## 動作環境

* 電車でDが動くコンピュータであること
* OS: Windows 10 64bit の最新のアップデートであること
* OSの端末が日本語に対応していること

※ MacOS 、 Linux などの Unix 系 OS での動作は保証できない。
